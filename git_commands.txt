git configuration commands:

1.
syntax:git config --global user.name
purpose:it sets your name in git.
        this name appears in your commits.

2.
syntax:git config --global user.email
purpose:sets your email address for commits.
        github connents commits to your account using email

3.
syntax:git config --list
purpose:shows all your git configuration settings

4.
syntax:git config --unset user.name
purpose:removes a configuration value
